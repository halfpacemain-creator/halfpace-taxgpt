**Steps:**
1. Public search on ipindiaonline.gov.in across the 45 Nice classes.
2. File **Form TM-A** — fees: ₹4,500 (individual / startup / MSME); ₹9,000 (others), per class per mark, e-filed.
3. Examination report issued; respond within 30 days, attend hearing if required.
4. Acceptance → publication in **Trade Marks Journal**; 4-month window for opposition.
5. If unopposed (or opposition decided in favour), registration certificate is issued.

**Validity:** 10 years from date of application; renewable in 10-year blocks via Form TM-R (fee ₹9,000 / ₹18,000 e-/physical, per class).
